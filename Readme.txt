1. download rufus to create a default linux bootable usb key

2. boot on the usb key and download diskwipepro from this page

3. extract files onto the desktop of the usb key from the .zip archive

4. right-click on the nist-wipe.sh file and click on allow running

5. double-click on the nist-wipe.sh file to run it 

6. follow the interactive menu's steps shown in the terminal and erase your disk according to your needs (ie pressing the number key that correspond to the disk you want to erase in the list of disk shown in the termimal texual menu) and wait for the erasure to finish

7. when you are done with erasing your disk, initalitize the disk in a disk manager and format it with the file system of your choice.

8. enjoy your gov. level erased disk ready for re-use.
